# Contribuindo para Aeternis Public Scripts

Obrigado por seu interesse em contribuir para este projeto! Como um repositório de scripts públicos, valorizamos a colaboração da comunidade para tornar esses scripts mais seguros, eficientes e úteis para todos.

## Como Contribuir

1.  **Reportando Bugs**: Se você encontrar um erro ou vulnerabilidade nos scripts, abra uma *Issue* descrevendo o problema.
2.  **Sugerindo Melhorias**: Tem uma ideia de como otimizar um script existente ou adicionar novas funcionalidades? Abra uma *Issue* para discussão.
3.  **Enviando Pull Requests**:
    *   Faça um Fork do repositório.
    *   Crie uma branch para sua modificação (`git checkout -b feature/minha-melhoria`).
    *   Certifique-se de que o script foi testado em um ambiente seguro.
    *   Faça o commit de suas alterações (`git commit -m 'Adiciona funcionalidade X'`).
    *   Faça o push para a branch (`git push origin feature/minha-melhoria`).
    *   Abra um Pull Request.

## Padrões de Código

*   Tente manter os scripts compatíveis com as distribuições Linux mais comuns (foco principal em Ubuntu/Debian).
*   Use comentários claros em português (ou inglês, se preferir manter o padrão).
*   Siga as boas práticas de segurança (evite expor senhas em texto plano, use `set -e`, etc).
